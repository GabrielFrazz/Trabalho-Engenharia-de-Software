Documentação da API - Gerenciamento de Clientes e Vendas

Base URL: http://localhost:5000/api

Endpoints Disponíveis

1. Listar Clientes

Rota: /clientes

Método: GET

Descrição: Retorna uma lista de todos os clientes cadastrados.

Exemplo de Requisição em Python:

import requests

url = 'http://localhost:5000/api/clientes'
response = requests.get(url)
clientes = response.json()

print(clientes)  # Exibe a lista de clientes

2. Adicionar Cliente

Rota: /clientes

Método: POST

Descrição: Adiciona um novo cliente à base de dados.

Formato do Corpo da Requisição (JSON):

{
    "nome": "Carlos Pereira",
    "email": "carlos@gmail.com",
    "telefone": "1122334455",
    "endereco": "Rua dos Testes, 78"
}

Exemplo de Requisição em Python:

import requests

url = 'http://localhost:5000/api/clientes'
novo_cliente = {
    "nome": "Carlos Pereira",
    "email": "carlos@gmail.com",
    "telefone": "1122334455",
    "endereco": "Rua dos Testes, 78"
}

response = requests.post(url, json=novo_cliente)

print(response.json())  # Exibe a resposta da API

3. Atualizar Cliente

Rota: /clientes/<int:id>

Método: PUT

Descrição: Atualiza as informações de um cliente existente, com base no ID fornecido.

Formato do Corpo da Requisição (JSON):

{
    "nome": "Carlos Pereira",
    "email": "carlos@novoemail.com",
    "telefone": "9988776655",
    "endereco": "Rua dos Atualizados, 99"
}

Exemplo de Requisição em Python:

import requests

url = 'http://localhost:5000/api/clientes/1'
dados_atualizados = {
    "nome": "Carlos Pereira",
    "email": "carlos@novoemail.com",
    "telefone": "9988776655",
    "endereco": "Rua dos Atualizados, 99"
}

response = requests.put(url, json=dados_atualizados)

print(response.json())  # Exibe a resposta da API

4. Deletar Cliente

Rota: /clientes/<int:id>

Método: DELETE

Descrição: Remove um cliente da base de dados com base no ID fornecido.

Exemplo de Requisição em Python:

import requests

url = 'http://localhost:5000/api/clientes/1'  # Substitua o ID pelo cliente desejado

response = requests.delete(url)

print(response.json())  # Exibe a resposta da API

Erros Comuns

  • 404 Not Found: O recurso (cliente ou venda) solicitado não foi encontrado no banco de dados.
  • 400 Bad Request: Dados inválidos ou incompletos foram fornecidos na requisição.
  • 500 Internal Server Error: Ocorreu um erro no servidor durante o processamento da requisição.
