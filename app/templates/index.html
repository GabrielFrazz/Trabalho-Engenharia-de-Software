Description

The project aims to develop financial management software for stores, facilitating the monitoring of debts, profits, sales, and customers. It includes features such as payment scheduling, graph generation, social media integration, loyalty program control, and customer feedback.

MAIN FUNCTIONALITIES OF THE PROJECT

Client Registration: Allows creating, viewing, editing, and deleting clients with forms, APIs, and notifications, including pagination and filters.

Sales Management: Provides sales recording, viewing, editing, and deletion with dedicated APIs, notifications, and filtering options.

Graph Generation: Creates interactive graphs for sales and profits, with filtering and data export options, using an API for aggregated data.

Payment Dates Management: Includes event scheduling, integration with payment systems, and cash monitoring, with financial reports and notifications.

Feedback Management: Collects and manages client feedback, linking it to each client, with notifications and options to export responses.
